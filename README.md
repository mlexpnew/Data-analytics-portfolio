Project Title

Customer Product Preference Ranking Dashboard

Problem

Survey data stored in wide format made comparison difficult.

Approach

• Melted dataset into long format
• Extracted product-metric pairs
• Created custom metric ordering
• Sorted to preserve logical structure
• Generated product insight

Result

Identified expectation gaps across products.
